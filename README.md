# aurelia webpack electron
⚠️ no longer maintained

development (hot module reload)

```
npm run develop
```

debugging (vscode): 
```
npm run debug
```
then press f5. once you are done you'll need to terminate the webpack-watch using `ctrl+C` in the terminal
